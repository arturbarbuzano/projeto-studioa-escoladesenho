# Projeto | StudioA - Academia de Desenho

O Studio A é um sistema desenvolvido com Spring Boot, Thymeleaf, HTML, CSS e PostgreSQL, criado para apresentar os cursos da Academia de Desenho Studio A, com foco em Cartoon, Mangá e Realismo. A escola tem como propósito ensinar os fundamentos do desenho e as técnicas específicas de cada estilo artístico, oferecendo uma abordagem completa para quem deseja evoluir na arte.

OBS: Falta a integração com o banco de dados PostgreSQL.
