# Projeto | StudioA - Academia de Desenho

O Studio A é um sistema CRUD desenvolvido com Spring Boot, Thymeleaf, HTML, CSS e PostgreSQL, criado para apresentar os cursos da Academia de Desenho Studio A, com foco em Cartoon, Mangá e Realismo. Basicamente, o site apresenta os três cursos oferecidos pela escola de desenho no qual a pessoa interessada entrará na página de Inscrição e colocará seu nome, CPF e técnica desejada. Depois disso, é só apertar no botão "Inscrever-se" e automaticamente ela estará cadastrada no sistema. Para ver a lista de usuários cadastrados, basta clicar na logo da escola que abrirá a página de lista de alunos. Nela, é possível editar alunos, assim alterando o nome, CPF e técnica do indivíduo desejado, além de poder deletar o usuário caso haja necessidade.

Link do site: https://projeto-studioa-escoladesenho.onrender.com
